simple rust file downloader using reqwest

## arguments
```
  -c, --content                Shows files in directory and closes program
  -l, --link <LINK>            Sets custom domain [default: https://static.klipy.com/ii/4e7bea9f7a3371424e6c16ebc93252fe/84/ef/1ocFw0eIBjDcaP.gif]
  -f, --filename <FILENAME>    Sets custom output file [default: file.bin]
  -d, --directory <DIRECTORY>  Sets custom output directory [default: downloads]
  -h, --help                   Print help
  -V, --version                Print version
```
